# `governance/working-groups/` — Sector Working Group charters

Each Sector Working Group has a charter document that describes its
scope, membership, and how it operates.

## Planned charters

One file per sector:

- `trade-swg.md`
- `logistics-swg.md`
- `mobility-swg.md`
- `hospitality-swg.md`
- `finance-swg.md`
- `services-swg.md`

## What each charter will contain

A typical SWG charter is expected to cover:

1. **Sector scope** — which transactions fall under this SWG (linked
   back to the sector principle in Annex A)
2. **Membership** — composition, eligibility, term length, rotation
3. **Operational scope** — what the SWG decides on its own and what
   it elevates to Council
4. **Working method** — meeting cadence, public participation, voting
5. **Sector-specific responsibilities** — the schema packs, policies,
   patterns, and category lists this SWG owns
6. **Engagement with regulators** — primary regulator(s) for the
   sector and how the SWG interfaces with them

## What is here today

Empty. Charters will be added as the working groups form.
